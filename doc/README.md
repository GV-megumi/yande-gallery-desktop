# 文档导航

`doc/` 目录现在分成三层：主文档、专题注意事项、历史归档。

## 主文档

- `doc/架构总览.md`：当前架构、模块分层、存储布局
- `doc/功能总览.md`：当前功能面
- `doc/开发与配置指南.md`：安装、命令、配置、常见排障
- `doc/数据库结构文档.md`：数据库结构总览
- `doc/Renderer API 文档.md`：渲染进程可调用的 `window.electronAPI` 总览
- `doc/Booru功能实现文档.md`：Booru 子系统说明
- `doc/图库功能文档.md`：图库子系统说明
- `doc/版本发布打包规范.md`：版本号、安装包、发布产物和 GitHub Release 流程

## 专题说明

- `doc/注意事项/README.md`：设计原因、约束、坑点索引

## 历史归档

- `doc/done/README.md`：已完成任务、历史方案、修复记录的归档入口

## 说明

- 如果你想了解“现在这个项目是什么样”，先看主文档。
- 如果你想了解“为什么要这样设计、哪些地方不能乱改”，看 `doc/注意事项/README.md`。
- 如果你想追溯某一轮功能实现或历史方案，再进入 `doc/done/README.md`。
