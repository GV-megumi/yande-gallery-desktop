# 功能总览

## 当前功能范围

本项目现在已经不只是早期的 Yande.re 浏览器，而是一个覆盖本地图库、Booru 多站点、批量下载、辅助云接入的桌面应用。

## 本地图库

### 已有能力

- 最近图片：懒加载浏览。
- 全部图片：分页浏览、按文件名或标签搜索。
- 图集管理：按目录组织，支持封面设置。
- 标签系统：标签增删、搜索、批量关联。
- 缩略图系统：WebP 缩略图缓存。
- 预览能力：缩放、拖拽、键盘导航。

### 相关模块

- 页面：`src/renderer/pages/GalleryPage.tsx`
- 服务：`src/main/services/imageService.ts`、`src/main/services/galleryService.ts`、`src/main/services/thumbnailService.ts`

## Booru 多站点

### 站点适配

- Moebooru
- Danbooru
- Gelbooru

### 基础浏览与搜索

- 帖子列表、单帖详情、分页与搜索。
- 热门帖子、Pools、标签详情页。
- 标签自动补全、高级搜索语法、高级过滤器。
- 随机帖子、相关标签推荐。

### 深化浏览能力

- 艺术家页、角色页、Wiki 页面、论坛浏览、用户主页。
- 帖子注释覆盖层、帖子版本历史。
- 视频帖子支持。
- 高级图片查看器：旋转、翻转、对比模式、元数据查看。

### 收藏与组织

- 本地收藏。
- 服务端收藏。
- 收藏夹分组。
- 收藏标签、标签分组。
- 黑名单标签。
- 保存的搜索。
- 服务端收藏、投票、评论、举报（按站点能力区分）。

### 下载与缓存

- 单帖下载。
- 下载队列：暂停、恢复、重试、状态查看。
- 批量下载任务与会话管理。
- 缓存统计与清理。
- 文件名模板系统。
- 数据备份与恢复。

### 其他高级能力

- 标签关系（alias / implication）浏览。
- 图片元数据读取。
- 论坛主题与帖子浏览。
- 帖子注释（notes）和版本历史。

### 相关模块

- 页面：`src/renderer/pages/Booru*.tsx`
- 服务：`src/main/services/booru*.ts`、`downloadManager.ts`、`bulkDownloadService.ts`、`imageCacheService.ts`、`backupService.ts`

## Google / 应用扩展区

- Google Drive 页面：通过 webview 嵌入 drive.google.com。
- Google Photos 页面：通过 webview 嵌入 photos.google.com。
- Gemini 页面：通过 webview 嵌入 gemini.google.com。

这部分能力与 Booru、本地图库并列挂在应用主导航中，属于辅助集成功能。所有 Google 页面均采用 webview 嵌入方式，不依赖 Google API 或 OAuth 认证。

## 设置 / 关于

- 设置页的"关于" Tab 有"检查更新"入口，拉取 GitHub Releases API 的最新版本并对比（v0.0.2 起）。成功结果在主进程做约 60s 的短时缓存，错误不缓存以便重试。
