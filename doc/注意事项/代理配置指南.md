# 代理配置指南

## 适用场景

- 所在网络环境无法直接访问 Google 或部分 Booru 站点。
- 需要统一代理主进程发出的 HTTP 请求。

## 配置方式

```yaml
network:
  proxy:
    enabled: true
    protocol: http
    host: 127.0.0.1
    port: 7890
```

## 原则

- 代理配置应由主进程读取并应用。
- 代理问题先检查配置，再检查代理服务是否运行。

## 不建议的做法

- 在页面层自行拼接代理 URL。
- 让不同服务各自维护不同代理配置入口。
