# Yande Gallery

桌面端（Electron）本地图库 + Booru 下载管理，安卓端为其局域网只读伴侣（镜像浏览 + 本机相册管理）。本词汇表固定跨端中文术语，尤其是「桌面域 / 手机域」两套相册概念的边界。

## Language

### 桌面域（数据源在桌面端，安卓经同步镜像）

**相册**:
桌面图库中的一个图片集合（gallery），安卓端经元数据同步镜像到 Room。
_Avoid_: 图集

**照片**:
安卓端展示全部桌面域图片的时间轴 tab。
_Avoid_: 时间轴（指代 tab 时）

**图片镜像**:
安卓 app 私有目录里的桌面图片文件副本，分高质量（HQ）与原图两档，`image_files` 表登记。
_Avoid_: 缓存、下载副本

**高质量图（HQ）**:
桌面端生成的长边 2560px 同格式压缩档，体积恒不大于原图，是手机端默认落盘档位。
_Avoid_: 预览图（那是已停用的 1600px 档）

### 手机域（数据源在本机 MediaStore，与桌面域完全隔离）

**手机相册**:
本机 MediaStore 的一个 bucket（由文件所在文件夹派生），在「手机相册」tab 展示与管理。
_Avoid_: 本地相册、系统相册

**本机媒体**:
MediaStore 里的一张图片或一条视频，手机域网格的基本单元。
_Avoid_: 本地图片

**全部照片**:
手机相册 tab 顶部的全库聚合视图（所有 bucket 的图片+视频合并时间倒序）。

**待落地相册**:
已命名但尚无任何文件的手机相册占位（仅本地记录），首个文件复制进入后由真实 bucket 接管。
_Avoid_: 空相册

### 跨域动作

**复制到**:
源保持不动、目标出现一份：目标为相册（桌面域）时是成员添加；目标为手机相册时是文件复制落盘。
_Avoid_: 加入相册（旧称）、导出、保存到相册

**移动到**:
源消失、目标出现：桌面域内是相册成员转移（目标仅限桌面相册，禁止移动到手机相册）；手机域内是文件物理移动到另一 bucket。

**移出当前相册**:
仅解除图片与当前桌面相册的成员关系，不删除图片，也不涉及任何目标相册。
