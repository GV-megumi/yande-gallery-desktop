> **执行时机：全部里程碑（M4）完成后统一联调。** 本文档只做计划存档，联调时按节执行并填写结果记录。

| 档位 | API | 必测项 | 说明 |
|------|-----|--------|------|
| 主力机 | 33+（Android 13+） | 全部 A-J 节 | POST_NOTIFICATIONS 运行时提示属 M4 项（M3 无前台通知），见 §J |
| 中间档 | 29-32 | D 节（scoped storage IS_PENDING 路径）、**F.11/F.12（29 直删级联分支）**、G 节 | RELATIVE_PATH/IS_PENDING 分支；29 无 createDeleteRequest |
